
# Disney Clone

A brief description of what this project does and who it's for


## Run Locally

Clone the project

```bash
  git clone https://github.com/codefundamentals1/disneyclone.git
```

Go to the project directory

```bash
  cd disneyclone
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run start
```
